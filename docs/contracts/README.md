# Contracts Docs

Контракты между backend, frontend, Kafka, платежами и внешними клиентами.

Use this folder for:

- REST/OpenAPI contracts;
- frontend/backend payload agreements;
- Kafka topics and outbox contracts;
- payment integration notes.

Key files:

- `API_CONTRACT.md`
- `FRONTEND_BACKEND_CONTRACTS.md`
- `KAFKA_TOPICS.md`
- `KAFKA_OUTBOX.md`
- `TELEGRAM_STARS_PAYMENTS.md`

If a contract changes behavior of a scenario, also update `docs/FSM_SCENARIOS_VIEWER.html` and the relevant file in `docs/fsm/`.
