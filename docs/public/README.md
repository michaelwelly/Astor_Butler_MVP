# Public Docs

Public/external docs package.

This folder stores public-facing Markdown notes that are not direct GitHub Pages HTML entrypoints.

Important root-level public URLs stay in `docs/` for compatibility:

- `docs/guest-guide.html`
- `docs/policy.html`
- `docs/staff-guide.html`
- `docs/LICENSE`

Do not move those HTML files unless the public links are updated at the same time.
