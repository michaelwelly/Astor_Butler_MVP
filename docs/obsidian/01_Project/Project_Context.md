# Project Context

## Коротко

Astor Butler MVP - Java 21 + Spring Boot монолит для Telegram-бота с FSM-архитектурой.

Telegram - транспорт и UI. FSM - single source of truth для сценариев общения.

## Продуктовая идея

AI-менеджер для ресторана или event-площадки, который ведет первичное общение с клиентом по бронированию даты и организации мероприятия:

- собирает вводные по событию;
- уточняет дату, формат, гостей, бюджет, меню и ограничения;
- фиксирует договоренности;
- готовит структурированную информацию для живого менеджера;
- помогает собрать документы и данные для бронирования.
