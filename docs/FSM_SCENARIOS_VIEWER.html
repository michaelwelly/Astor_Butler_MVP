1. Общий вход

Все сообщения проходят через MessageGatewayService. Админ-чат не является бизнес-сценарием гостя: он сохраняется и используется для наблюдаемости.

Открывай этот файл в браузере: Cmd + / Cmd - масштабируют диаграммы.
