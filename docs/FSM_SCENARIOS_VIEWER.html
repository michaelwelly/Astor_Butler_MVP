18. Контракт LLM

LLM - адаптер языка. Он помогает понять и красиво сказать, но не владеет бизнес-решениями.

LLM можетКлассифицировать intent, извлекать дату/время/гостей/зону/сумму/лот, задавать одно уточнение, помогать с тоном.
LLM не можетПодтверждать бронь, создавать holds, принимать ставки без подтверждения, раскрывать приватные донаты, менять статус заявки.
Формат будущего результатаintent, confidence, entities, needsClarification, clarificationQuestion.
